# 자기도출과 확신의 구조

**Self-Derivation and the Structure of Conviction: How Conviction Emerges Beyond Justification**

---

## 문제

뮌하우젠 트릴레마는 궁극적 정당화가 불가능함을 보여준다. 그러나 인류의 절반 이상이 특정 믿음 체계에 깊은 확신을 유지한다. 정당화 없이 확신은 어떻게 창발하는가?

## 핵심 개념

1. **자기도출(Self-Derivation)**: 이론의 *내용*($M_{\text{subject}}$)이 참이라면 이론의 *존재*($M_{\text{object}}$)가 정합적으로 기대되는 구조. 아브라함 계열 유신론에서 "신이 존재한다면 계시가 있을 것이다"라는 추론이 이 구조를 예시한다.

2. **내용-존재 정합성(Content-Existence Coherence)**: 자기도출의 핵심 조건. $M_{\text{subject}}$가 참이라면 $M_{\text{object}}$의 존재가 설명적으로 자연스럽다는 것. 연역적 함축이 아니라 개연적 정합성이다.

3. **준-정당화(Quasi-Justification)**: 자기도출이 산출하는 심리적 효과. 정당화가 아니지만, 정당화*처럼* 확신을 창발시킨다—quasi-particle이 입자가 아니면서 입자*처럼* 기능하듯이.

## 발견

1. **확신의 구조적 조건**: 자기도출 → 준-정당화 → 확신. 이 메커니즘이 정당화 없이 확신이 창발하는 구조다.

2. **자기도출 트릴레마**: 단순성, 외부 설명력, 자기도출 사이에는 구조적 긴장이 존재한다. 과학은 자기도출을 유예함으로써 설명력을 얻었고, 종교는 설명력을 희생함으로써 자기도출을 얻었다.

## 요약

> 완벽한 토대는 없다. 그러나 확신이 어떤 구조 위에서 작동하는지는 해명할 수 있다.
