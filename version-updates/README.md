# Version Updates

논문 「자기도출의 트릴레마」 버전 관리 기록.

## 버전 규칙

| 유형 | 표기 | 기준 | 예시 |
|:---|:---|:---|:---|
| **Major** | X.0 | 제목 변경 | 1.0 → 2.0 |
| **Minor** | X.Y | 피어리뷰 반영 | 1.1 → 1.2 |
| **Patch** | X.Y.Z | 자체 수정 | 1.2 → 1.2.1 |

## 변경 이력

| 버전 | 날짜 | 유형 | 요약 |
|:---|:---|:---|:---|
| 1.1 → 1.2 | 2025-02 | Minor | 피어리뷰 반영: 준-정당화를 기술적 개념으로 재정의, 범위를 아브라함 계열 유신론으로 한정, 트릴레마를 "구조적 긴장"으로 약화 |

## 파일 구조

각 버전 업데이트는 `[이전버전]-[새버전].md` 형식으로 기록한다.

- **포함 내용**: 리뷰어 지적사항, 수정 전략, 상세 변경 내역, 리뷰 결과
- **목적**: 수정 근거 추적, 유사 지적 재발 시 대응 자료
