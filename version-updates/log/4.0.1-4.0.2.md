# 4.0.1 → 4.0.2

**날짜**: 2025-12
**유형**: Patch (자체 수정)
**근거**: 자기도출 정의 정밀화 - 반례 선제 차단 및 개념적 경계 명확화

---

## 변경 요약

"인간은 글을 쓴다"와 같은 trivial한 자기참조가 자기도출로 오인되지 않도록 정의를 정밀화함. 자기도출의 두 조건(설명적 포괄성, 연속적 도달)을 명시하고, 자기참조/일반 과학/자기도출의 구분표 추가.

---

## 주요 변경 사항

### 1. 3.2절 정의 구조 개편

**변경 전**: 내용-존재 정합성만으로 자기도출 정의
**변경 후**:
- 내용-존재 정합성을 먼저 정의
- "인간은 글을 쓴다" 반례를 선제적으로 다룸
- 자기도출의 두 조건 명시:
  1. **설명적 포괄성**: $M$이 자기 존재 외에도 광범위한 현상을 설명
  2. **연속적 도달**: 설명 경로의 연장선에서 자기 존재에 도달

### 2. 구분표 추가 (3.2절)

| | 외부 설명력 | 자기 포함 | 구조 |
|---|---|---|---|
| "인간은 글을 쓴다" | ✗ | ✓ | 자기참조 |
| $F=ma$ | ✓ | ✗ | 일반 과학 |
| 아브라함 유신론 | ✓ | ✓ | **자기도출** |
| 헤겔 체계 | ✓ | ✓ | **자기도출** |

### 3. "도출"(derivation) 용어 정당화

- 참조(reference): 직접 가리킴
- 도출(derivation): 경로를 따라 도달함
- 자기도출은 외부 설명 경로의 연장선에서 자기 존재에 "도달"하므로 "도출"이 적합

### 4. 수식 추가

$$M_{\text{subject}} \xrightarrow{\text{외부 설명}} \cdots \xrightarrow{\text{연속적 도달}} M_{\text{object}}$$

### 5. 3.5.2절 자기참조 설명 보강

- 개정된 정의와 연결
- "설명적 포괄성 부재"를 자기참조의 핵심 특징으로 명시
- 3.2절의 "인간은 글을 쓴다" 반례 참조

### 6. 7.1절 한계 보완

- "설명적 포괄성"의 정량화가 미해결 과제임을 명시
- "얼마나 광범위해야 충분한가"에 대한 정량적 기준 부재 인정

### 7. 설명적 포괄성 조건 단순화

**변경 전**: "광범위한 현상을 설명하거나, 삶의 근본 물음(기원, 목적, 죽음)에 답한다"
**변경 후**: "광범위한 현상을 설명한다"

"삶의 근본 물음" 조건 삭제 - 헤겔 등 비종교적 사례에 과적합되었던 표현 제거

## 논문에의 기여

1. **개념적 경계 확립**: 자기도출이 단순 자기참조와 어떻게 다른지 명확히 함
2. **반례 선제 차단**: 리뷰어가 생각할 법한 반례를 저자가 먼저 제시하고 해소
3. **"도출" 용어 정당화**: 왜 "자기참조"가 아닌 "자기도출"인지 설명
4. **사례 중립성**: 종교와 철학(헤겔) 모두에 적용 가능한 정의
