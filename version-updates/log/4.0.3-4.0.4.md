# 4.0.3 → 4.0.4

**날짜**: 2025-12
**유형**: Patch (자체 수정)
**근거**: 5.6절 신설 (변증법적 자기도출의 계승), 초록 수정, 용어 일관성

---

## 변경 요약

헤겔 → 마르크스 연결을 명시화하여 헤겔 분석의 현대적 relevance 확보. 자기도출의 두 전략(구체화 vs 추상화) 대조 추가. 20세기 양대 확신 체계(기독교, 공산주의)가 동일한 구조적 조건을 충족함을 시사.

---

## 주요 변경 사항

### 1. 5.6절 신설: "자기도출의 두 전략: 구체화와 추상화"

**추가 내용**:
- 종교는 경로를 **구체화**: 서사적 사건들 (창조, 타락, 계시)
- 변증법은 경로를 **추상화**: 논리적 범주들 (존재, 본질, 개념)
- 대조표 추가

| | 종교 | 변증법 |
|---|---|---|
| 전략 | 경로의 구체화 | 경로의 추상화 |
| 경로 구성 | 서사적 사건들 | 논리적 범주들 |
| $M_{\text{object}}$ 도출 | "왜 이 경전이 있는가" | "왜 이 체계가 있는가" |

### 2. 마르크스의 "합리적 핵심" 인용

- Marx, *Capital* Vol. I, Afterword to the Second German Edition (1873)
- "With him [Hegel] it is standing on its head. It must be turned right side up again, if you would discover the rational kernel within the mystical shell."
- 변증법적 총체성 구조가 역사유물론에 보존됨

### 3. 20세기 양대 확신 체계

- 기독교와 공산주의가 각각 구체화와 추상화라는 상이한 전략으로 동일한 구조적 조건을 충족
- 자기도출이 확신의 가능조건임을 시사

### 4. 초록 수정

**변경 전**: "19세기 헤겔주의가 보인 교조적 추종과 분파 형성은 이 구조의 효과를 시사한다"
**변경 후**: "이 구조는 마르크스의 역사유물론에 계승되었으며, 20세기 공산주의가 보인 확신의 양상은 이 구조의 효과를 시사한다"

**근거**:
- "19세기 헤겔주의의 교조적 추종"은 학계 내 현상으로 일반 독자에게 낯설고 검증 어려움
- 공산주의의 확신적 양상은 20세기 역사에서 잘 문서화됨

### 5. 결론 용어 일관성

**변경 전**: "그 확신에 대한 열림의 가능조건이다"
**변경 후**: "그 확신에 대한 비판적 거리의 가능조건이다"

**근거**: "열림"은 ver3.x의 잔재. 초록/서론/결론에서 "비판적 거리"로 통일.

## 변경 판단 근거

| 기준 | 판단 |
|:---|:---|
| 현대적 relevance | ◎ (헤겔 → 공산주의 연결로 독자 이해 용이) |
| 증거 강도 | ◎ (공산주의 확신은 역사적으로 잘 문서화) |
| 구조적 분석 | ◎ (구체화 vs 추상화 대조로 자기도출 유형 명확화) |
| 용어 일관성 | ◎ (초록-서론-결론 통일) |

---

## 논문에의 기여

1. **헤겔 분석의 정당화**: "왜 헤겔?" → 공산주의의 구조적 토대
2. **양대 확신 체계 병치**: 기독교 + 공산주의가 같은 조건 공유
3. **자기도출 유형론**: 구체화/추상화라는 두 전략 제시
4. **증거 강화**: 학계 현상(헤겔주의) → 대규모 역사적 현상(공산주의)
