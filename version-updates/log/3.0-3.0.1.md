# 3.0 → 3.0.1

**날짜**: 2025-12
**유형**: Patch (자체 수정)
**근거**: 제목/부제와 초록의 용어 일관성 확보

---

## 변경 요약

초록에서 "산출" → "창발"으로 변경, quasi-particle 유비 한 줄 추가.

---

## 주요 변경 사항

### 1. 초록 첫 문장 (45줄)

**이전**:
> 정당화가 불가능한 세계에서, 강력한 확신은 어떻게 **산출**되는가?

**이후**:
> 정당화가 불가능한 세계에서, 강력한 확신은 어떻게 **창발**하는가?

**근거**: 부제 "정당화 너머에서 확신은 어떻게 창발하는가"와 일관성.

### 2. 준-정당화 설명 (47줄)

**이전**:
> 자기도출은 정당화가 아니지만, 정당화와 유사한 심리적 효과---준-정당화---를 산출한다.

**이후**:
> 자기도출은 정당화가 아니지만, 정당화와 유사한 심리적 효과---준-정당화---를 창발시킨다. "Quasi-"는 물리학의 quasi-particle에서처럼 "존재론적으로 다르지만 기능적으로 동등함"을 뜻한다.

**근거**:
- "산출" → "창발"으로 용어 통일
- quasi-particle 유비를 초록에서 간략히 언급 (본문 3.4절과 연결)
