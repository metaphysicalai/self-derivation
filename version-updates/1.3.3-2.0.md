# 버전 업데이트 기록: 1.3.3 → 2.0

**문서**: 자기도출의 트릴레마
**업데이트 일자**: 2025-12-19
**업데이트 유형**: Major (제목 변경)

---

## 업데이트 개요

### 수정 유형
- **Major**: 제목 변경

### 변경 사유
- 기존 부제("뮌하우젠 트릴레마의 재구성")가 논문 내용과 불일치
- 논문은 뮌하우젠을 "재구성"하지 않고, 배경으로만 사용
- 핵심 주장("준-정당화", "정당화 너머의 확신")이 제목에 반영되지 않음
