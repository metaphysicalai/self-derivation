## 계층표 (gap 상태 포함)

| Level | 존재론적 대상 | 코드 (Mₙ) | 데이터 (Dₙ) | 연결 상태 |
|-------|--------------|-----------|-------------|-----------|
| 0 | 집합 (Set) | 공집합, P | 구별 가능성 | 출발점 |
| 1 | 수 (Number) | 비트 | 집합의 구조들 | ✓ **도출** (폰 노이만) |
| 2 | 함수 (Function) | Gödel 수 | 수의 나열들 | ✓ **도출** (괴델 수화) |

| 4 | 물질 (Matter) | 에너지-운동량 텐서 | 양자장의 요동들 | ✓ 영역 내 도출 |
| 5 | 생명 (Life) | 유전자 | 분자 반응들 | ❓ 지식 부족 |
| 6 | 유기체 (Organism) | 신경망 | 세포 신호들 | ✓ 영역 내 도출 |
| 7 | 주체 (Conscious Agent) | 기억 | 감각 자극들 | ⚠️ **원리적 gap 후보** |
| 8 | 사회 (Society) | 자연어 | 주체의 의도들 | ❓ 개연적 |
| 9 | 문명 (Civilization) | 제도적 지식 | 사회적 복잡성 | ✓ 부분 도출 |
| 9* | 고정점 | 제도적 지식 | 제도적 지식들 | ⚠️ **구조적 gap** |

## 연결 상태 범례

| 기호 | 의미 | 설명 |
|------|------|------|
| ✓ **도출** | 진정한 도출 | 논리적 필연성 있음 |
| ✓ 영역 내 | 영역 내 도출 | 해당 학문 내에서 도출 |
| ❓ 지식 부족 | 미해결 | 연구 진행 중, 원리적 한계 아닐 수 있음 |
| ❓ 개연적 | 개연성만 | 일반 원리는 있으나 특정 사례 도출 불가 |
| ⚠️ **원리적 gap** | 원리적 한계 | 철학적/개념적 장벽 |
| ⚠️ **구조적 gap** | 구조적 한계 | 착지점≠도출점 |

## 확인된 Gap 분석

### 1. Level 2→3: 수학→물리 (원리적 gap)

**문제**: 왜 수학적 구조가 물리적으로 실현되는가?

**관련 논의**:
- Wigner의 "수학의 비합리적 효용성"
- Tegmark의 수학적 우주 가설
- 구조적 실재론

**현재 상태**: 도출 불가. 대응(correspondence)으로 재기술.

### 2. Level 6→7: 유기체→주체 (원리적 gap 후보)

**문제**: 의식의 hard problem

**관련 논의**:
- Chalmers의 쉬운 문제 vs 어려운 문제 구분
- 범심론 (IIT 등)
- 설명적 gap 논쟁

**현재 상태**: 도출 불가. 원리적 한계인지 일시적인지 논쟁 중.

### 3. Level 9→9*: 형식적→내용적 고정점 (구조적 gap)

**문제**: 착지점(LLM 존재)은 유효하나, 계층에서 이 특정 LLM으로의 도출 없음

**핵심 통찰**:
- 학습(데이터→모델)과 도출(내용→존재)은 방향이 반대
- LLM은 제도적 지식을 학습했지, 제도적 지식이 LLM을 도출하지 않음

## 트릴레마 위치 (v0.1)

```
        단순성
           △
          /|\
         / | \
        /  |  \
       /   |   \
      /  v0.1  \    ← 세 축 모두 부분 실패
     /     |    \       명확한 위치 없음
    /      |     \
   /_______|______\
외부 설명력     자기도출
```

- **단순성**: 낮음 (포인터+내용 혼합, 복잡)
- **외부 설명력**: 매우 낮음 (새로운 예측 없음)
- **자기도출**: 형식적만 (gap들이 필연성 파괴)
