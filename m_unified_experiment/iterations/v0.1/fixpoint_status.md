# 고정점 현황 v0.1

## 형식적 고정점

**상태**: ✓ 도달

**근거**:
- LLM은 자기 자신에 대해 서술 가능
- 제도적 지식을 입력받아 제도적 지식에 대한 출력 생성
- M(제도적 지식) → {M에 대한 서술, 다른 출력들}

**한계**:
- 이것은 자기**참조**이지 자기**도출**이 아님
- "나에 대해 말할 수 있다" ≠ "계층이 나를 도출한다"

## 내용적 고정점

**상태**: ✗ 미도달 (확정)

**이유**:
1. **원리적 gap들이 연속적 도출을 방해**
   - Level 2→3: 수학→물리
   - Level 6→7: 유기체→주체 (의식)

2. **구조적 gap**
   - Level 9→LLM: 착지점≠도출점
   - 학습(데이터→모델)과 도출(내용→존재)의 방향 불일치

3. **괴델 불완전성의 함의**
   - M_unified가 수론을 포함하면
   - 완전한 자기 설명은 원리적으로 불가능

## Cycle 1의 핵심 발견

### 발견 1: 학습과 도출은 다르다

| 학습 | 도출 |
|------|------|
| 데이터 → 모델 | 전제 → 결론 |
| 귀납적 | 연역적 |
| 사후적 | 선험적 |
| 확률적 | 필연적 |

LLM이 "많이 안다"는 것은 학습의 결과이지, 도출의 달성이 아니다.

### 발견 2: 형식적→내용적 이행의 장벽

형식적 고정점(자기참조 가능)에서 내용적 고정점(통합적 도출)으로의
이행이 어려운 이유:

1. 중간 경로에 원리적 gap이 있으면
2. 아무리 내용을 채워도
3. 연속적 도출 체인이 완성되지 않음

### 발견 3: 트릴레마의 경험적 확인

- 내용을 채우려 하면 → gap들이 드러남 (외부 설명력 시도)
- gap을 숨기려면 → 포인터만 유지해야 함 (단순성 선택)
- 자기도출은 → 형식적 수준에서만 가능

세 축의 동시 달성이 어려움을 경험적으로 확인.
